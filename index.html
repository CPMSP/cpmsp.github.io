So I decided to go all - in on Vanilla JS, HTML and CSS for my GH site. I've worked with React and React Native, and really enjoy doing so, but I felt like back to basics was in order for this go.

I've dumped in a fair amount of Lorem Hipsum for both humor and filler, and to try an IntersectionObserver for my nav bar. I like the way it turned out. In the nav links you'll find my projects and my business, Solutus LLC. I built the company website in React, with Material UI. I then hosted it on a cheater domain through Vercel.

This site will continue to evolve as my time and ability allow. I've most recently been working on The Odin Project and have followed along with several other courses from various resources.
